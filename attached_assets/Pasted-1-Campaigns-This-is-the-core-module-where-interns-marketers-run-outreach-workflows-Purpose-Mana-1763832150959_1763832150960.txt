1. Campaigns

This is the core module where interns/marketers run outreach workflows.

Purpose

Manage uploaded lead files, generate insights, perform enrichment (email/website finding), and track outreach tasks.

Key Features

Upload Excel/CSV

Run Gemini-based insight generation

Lead enrichment (email + website finder)

View all previously generated insight batches

Export insights (PDF, DOCX, CSV)

“Find Email” and “Find Website” options per lead

Status indicators (Processed, Pending, Need Review)

Subpages

New Campaign (Upload file + run analysis)

Campaign History

Lead Insights View

Email Finder Panel

Export Center

UI Notes

Campaign cards showing: Name, Date, File uploaded, Leads count

CTA: “Create New Campaign”

Clean table/list layout for leads

2. Templates

Features

Edit + save custom templates

One-click “Copy message”

Integration: Insert template in Campaign > Lead view

Subcategories

Conversation starters

Cold email templates

LinkedIn outreach messages

Follow-up messages

Client explanation scripts

UI Notes

Template cards with title + preview

Rich-text editor for customization
